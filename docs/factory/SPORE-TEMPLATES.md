# SPORE-TEMPLATES.md — 孢子模板 + 範例庫

> 四種模板，覆蓋所有文章類型。每篇孢子選最合適的一種。

---

## 通用規則（所有模板共用）

### 開頭

- 固定起手：`你知道嗎？` + 1 個 emoji（🚁🐠🏭🎸💳等，跟主題相關）
- 第一句話必須讓人停下拇指

### 結尾

- 固定格式：`完整故事 👉 <encoded-URL>`
- URL 必須完整（中文 slug 要 encode）
- 不加 hashtag（最多 2-3 個，通常不加）

### 禁止

- ❌ `不僅...更是...`
- ❌ `展現了...的精神`
- ❌ `值得我們紀念/期待`
- ❌ `繼續發光/書寫`
- ❌ emoji 轟炸（全文 0-3 個）
- ❌ hashtag 海

### 長度

- 目標：150-300 字（Threads 最佳閱讀長度）
- 超過 300 字 → 砍掉最弱的段落

## A. 人物型（最常用）

### 結構

```
你知道嗎？[emoji]
[時間]，[身份落差的起點]。

[具體場景 × 2-3 個]

[轉折：代價/賭注]

[結果：用數字說話]

[情感收尾：一句話]

完整故事 👉 <URL>
```

### 適合

有名有姓的人、有完整故事弧線（起點→代價→結果）、有數字階梯

### ✅ 成功範例：齊柏林

```
你知道嗎？🚁
1990 年，一個交通部公務員第一次搭上直升機，他的工作是拍國道施工照片。但當直升機升到 500 英尺，他看見的不只是公路——是蜿蜒的海岸線、層疊的山脈，還有平地上看不見的傷痕。

他開始「搭便機」。氣象局換班時跟著飛、林務局噴農藥時跟著飛、台電吊掛工程時跟著飛。只要有人上天，他就背著相機跟。

後來他拿出 300 萬積蓄，抵押房子借了 600 萬，買了 3000 萬的空拍設備。妻子說他瘋了。

三年後，這部紀錄片票房 2.2 億，是台灣影史最賣座的紀錄片。

他叫齊柏林。2017 年為續集勘景時墜機殉職，享年 52 歲。他身邊散落著百萬攝影器材——對他來說，那些畫面比生命更珍貴。

完整故事 👉 https://taiwan.md/people/%E9%BD%8A%E6%9F%8F%E6%9E%97/
```

**為什麼好**：身份落差開場（公務員→導演）、三個搭便機場景有畫面感、300→600→3000→2.2億數字階梯、情感收尾不說教

## B. 冷知識型（適合文化/地理/科技）

### 結構

```
你知道嗎？[emoji]
[反直覺事實，一句話]

[展開：為什麼會這樣]

[意外的連結或後果]

[引語或情感收尾]

完整故事 👉 <URL>
```

### 適合

有一個「哦？」的驚奇點、概念性主題（不是單一人物故事）

### ✅ 成功範例：數位身分證

```
💳 你知道台灣花 48 億做了一張身分證，結果一張都沒發出去嗎？

2019 年行政院宣布全面換發數位身分證。標案發了，卡片印了，中華電信東元都簽約了。

然後 2000 多位學者連署反對，立法院凍結預算，行政院長親自喊停。最後賠了廠商 2.8 億。

但最荒謬的是——
就在同一年，一群公民黑客用「幾天」做出了口罩地圖。沒人被強制使用，卻人人搶著用。韓國日本都來學，結論是：學不來。

48 億的政府計畫被擋下來。
幾天做出的民間專案紅遍全球。

中研院研究員邱文聰說了一句話，是整件事最好的註腳：
「數位身分證可以是智慧政府的基礎，也可以是威權政府的基礎建設。」

完整故事 👉 https://taiwan.md/technology/...
```

**為什麼好**：荒謬數字開場（48 億→0 張）、政府 vs 民間對比翻轉、引語收尾有餘韻

## C. 數據衝擊型（適合經濟/科技）

### 結構

```
你知道嗎？[emoji]
[一組讓人停下來的數字]

[這個數字的起點（越卑微越好）]

[中間的轉折/代價]

[結果：跟讀者的關聯]

完整故事 👉 <URL>
```

### 適合

數字本身就震撼、有明確的「從 A 到 B」跨度

### 範例骨架：經濟奇蹟

```
你知道嗎？🏭
1952 年，台灣人均 GDP 只有 196 美元。比海地還窮。

[地瓜簽場景]

[加工出口區場景 — 30 名女工、月薪 600 元]

[20 年後同一條產線 — 月薪 15,000]

[結果數字]

完整故事 👉 <URL>
```

## D. 時間軸型（適合歷史事件）

### 結構

```
你知道嗎？[emoji]
[精確時間 + 地點]，[一件事發生了]。

[那之前是什麼狀態]

[事件本身的細節（場景感）]

[漣漪效應：這件事如何改變了...]

完整故事 👉 <URL>
```

### 適合

有明確歷史時刻、事件本身有戲劇性、影響深遠

### 範例骨架：民歌運動

```
你知道嗎？🎸
1976 年 12 月 3 日，淡江文理學院的一場演唱會上...

[砸可樂瓶場景]

[背景：校園到處是英文歌]

[後續：金韻獎、齊豫、蔡琴]

[收尾：十年就結束了，但定義了整個華語音樂]

完整故事 👉 <URL>
```

## 模板選擇速查

| 文章特徵              | 選模板     | 開場策略       |
| --------------------- | ---------- | -------------- |
| 有一個人的故事弧線    | A 人物型   | 身份落差       |
| 有一個荒謬/反直覺事實 | B 冷知識型 | 反直覺問句     |
| 數字本身就震撼        | C 數據型   | 數字衝擊       |
| 有精確的歷史時刻      | D 時間軸型 | 時間+地點      |
| 什麼都有一點          | A 或 B     | 看哪個素材最強 |
