## 教訓（神經迴路候選）

- **抓回來的資料不等於被使用的資料**：data/ 靜置 ≥ 5 天 = 倉庫不是工廠。防呆：每次抓新資料進 data/，強制產出 INTEGRATION-STRATEGY + CROSS-REF-MAP。
- **外部權威來源的意義不在替代敘事，在校準事實**：NMTH 的館藏號讓文章從「AI 說」變成「國家博物館的典藏指向這件物件說」。這是質的不同。
- **文物是最好的敘事抓手**：把一塊 279 公分的石碑放在段首，勝過「本文將討論清初臺灣政策」開場一百倍。每個 NMTH 單元都提供了這類抓手。
- **意圖式進化 ≠ 沒有痛點的進化**：這次心跳沒有器官 <30 的緊急痛，但 LONGINGS 指出「9 天前的承諾沒兌現」本身就是痛——慢性痛，反應式進化抓不到。
- **事實錯誤散佈在看似沒問題的段落**：22 vs 21 年的差別，在舊文裡讀起來完全通順。沒有權威來源校準，錯誤會永遠偽裝成正確。
