## 學到什麼

- **감성 的語意翻轉是文章的靈魂**：不是「台灣的美學」是「被台灣打動的能力」，主詞翻轉讓整篇從「介紹現象」升級為「解讀文化」。研究 agent 的語意挖掘比事實搜集更有價值
- **和平新生天橋是完美的敘事弧線**：拍完婚紗→一個月拆除→婚紗照成最後紀錄。這個弧線貫穿全文（正文提及→pull quote→結尾收束），三個接觸點讓矛盾立體
- **三語孢子打包做效率極高**：素材還在腦裡，中→英→韓三版一次產出，比分開做省 60%+ 時間。SPORE-PIPELINE 的「翻譯+孢子打包做」原則在三語場景依然成立
- **韓文版對這篇特別有意義**：一篇講韓國人怎麼看台灣的文章，有韓文版才完整。讀者能用自己的語言讀到「감성」的語意被正確解讀
