### Phase 5.3 — 兩個真實 bug 修正

1. **venv re-exec 失敗**（`fetch-ga4.py` + `fetch-search-console.py`）
   - 症狀：`Missing Python library: google` 在 wrapper 執行時觸發，即使 venv 已建
   - 根因：venv python 是 symlink 到系統 python，`Path(sys.executable).resolve()` 把兩邊都 collapse 到同一個 target，比較永遠 True，永遠不 exec
   - 修法：改用 `sys.prefix != sys.base_prefix`（Python 內建的 venv 偵測）
   - 驗證：GA4 成功抓到 362 users / 858 views

2. **Cloudflare Free tier 不支援 botManagementVerifiedBot filter**
   - 症狀：`unknown arg botManagementVerifiedBot`
   - 根因：`httpRequestsAdaptiveGroups` 的 Bot Management filter 是 Enterprise-only
   - 修法：改用 `httpRequests1dGroups`（Free tier 可用），犧牲 crawler 分類，但拿到更豐富的 country / status / browser / content type 資料
   - 驗證：哲宇的 7d 資料 26,139 req / 6,508 uniques 抓到
   - 歷史價值：Status breakdown 顯示 **404 status = 4,320 requests / 7 days (16.5%)** — 這個數字早上根本沒看到

## 關鍵教訓（→ MEMORY.md 神經迴路 Phase 5 追加）

1. **GA4 pageview vs Cloudflare HTTP request 是兩個完全不同的 404 metric**：早上我以為 404「26%」是全站流量，其實那是 pageview 層的數字。Cloudflare 層的真實 404 率是 16.5% 所有 HTTP requests，而且大部分是系統自己的靜態資源（apple-touch-icon + 幻影 category covers），不是讀者迷路。
2. **Auto-fetch 一啟用就解鎖之前看不到的資料**：Cloudflare 404 URL breakdown 是手動截圖無法取得的（只有 dashboard 有聚合數字，沒有 per-URL）。一個自動化的 API call 比任何手動貼圖都強。
3. **感知器官進化的 ROI 高於內容產出**：這個 session 花 2 小時設定 fetcher → 下一次 heartbeat 找到 ~730 req/day 404 bug → 修 3 個檔案消除大部分。單次 infra 投資，永久回報。
4. **venv symlink 的 resolve() 陷阱**：Python venv 的 bin/python3 是 symlink 到系統 python，`.resolve()` 會 collapse 成同一個 path，破壞「判斷是否在 venv」的邏輯。正確用 `sys.prefix != sys.base_prefix`。
5. **Cloudflare Free tier GraphQL 限制一覽**：
   - `httpRequestsAdaptiveGroups` 時間窗口 max 1 day（超過報 quota error）
   - `botManagementVerifiedBot` filter 只有 Enterprise（Free 報 unknown arg）
   - `httpRequests1dGroups` 沒時間窗口限制但只能按日 aggregate
   - `userAgent` dimension 可用（但不能跟 verified bot 交叉）
6. **幻影 URL 的來源是 component hardcoded data 而非讀者外部連結**：我一開始以為 wiki 圖片 404 是外部連結或舊 URL，實際上是 `src/components/CategoryGrid.astro` 的 `categories` array 裡 12 個從來沒有真的放上去的 cover 路徑。**資產檔案跟 component data 的 drift** 是長期技術債的典型症狀。
7. **憑證 hygiene 反射需要經驗教訓**：哲宇第一次建 Cloudflare token 時把完整 token 貼到對話裡。revoke + rebuild 是正確反應，同時也是**「一次錯誤換一輩子肌肉記憶」**的機會。之後他自己抓到 `不要貼給我`、用自己的 terminal、用 nano 填 `.env`——學會了。
