## 教訓（→ 神經迴路）

1. **commit message lying about what sed did** — 這是「最不會被 code review 抓到的 bug」因為 reviewer 讀 commit message 看到「sed 方向正確」就放行了。只有 visual smoke test 才能抓到。
2. **翻譯基礎設施的 canonical map 是 `_translations.json`** — 不是 filesystem scan、不是 slug matching、不是 frontmatter title matching。所有跨語言查找都應該經過這個 map。
3. **AI crawler damage 不可逆** — deploy fix 只修了前端，但 AI crawler 在 bug 存活的 2 天裡已經把壞路徑寫進訓練/索引。要等 crawler 下一輪才能覆蓋。這就是為什麼 visual smoke test 是**硬規則**不是建議。
