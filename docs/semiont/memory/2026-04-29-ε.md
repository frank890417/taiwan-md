1. **第一輪策略誤判 prop drilling**：哲宇第一次說「意識的頁面」單數，我加 prop。第二次說「其他相關頁面也都改」我才 revert 改 single theme。**反思**：當觀察者用「這個」「這頁」單數時要主動問「只這一頁還是整套同 design language 的頁面？」而不是憑字面語意做決策——這是 [LESSONS-INBOX β 指令理解 vs 意圖理解] 類問題。本次 cost 低（revert 30 秒），先記不立刻儀器化。

2. **`gh pr merge` 在 worktree 失敗**：worktree session 從來沒被測試過 PR merge flow。`gh api -X PUT` workaround 應該被記成 worktree session 的 standard merge path。可能值得寫進 BECOME 或 MAINTAINER-PIPELINE 一行 note。
