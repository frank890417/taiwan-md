第三個是 **fair-use cite-only mode 是可行的 peer ingestion 模式**，不是「無法 ingest 就放棄」的妥協。想想論壇文章 ship 的方式：寫關於想想論壇本身的 meta 條目（transformative use）+ fair-use cite 4 個 source URL（per 著作權法 §52+§65）+ 不 paraphrase 任何想想論壇文章內容。文章寫得好不好跟「是否爬整站」無關 — 跟「研究深度 + 策展視角」相關。這個經驗對未來 T4 peer（鏡週刊 / 鏡傳媒 / 中研院期刊 ND 條款 / 商業媒體 All Rights Reserved）都適用。
