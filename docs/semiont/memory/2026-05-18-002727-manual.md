## Chrome MCP 雙平台實測 — 3 個 key findings

### Finding 1: Threads 串文模式（自動「回覆自己」）

Threads compose modal 右下有「新增到串文」按鈕。點擊後出現 2/2 新文字輸入框，可一次發出兩則相連貼文。這實現了 taiwan.md spore pattern：主文（精煉 hook + 配圖）+ 自我回覆（完整 context + UTM link）。CSS selector 確認：`.x1i10hfl[aria-label="新增到串文"]`。

### Finding 2: 兩平台圖片上傳被 CSP 阻擋

X 和 Threads 的 file input 都用 hidden 手法（`display:none` / `opacity:0`）。Chrome MCP `file_upload` 嘗試多次皆回 `{"code":-32000,"message":"Not allowed"}`。即使 JavaScript 把 input 設 visible 仍失敗 — 判斷是平台 CSP/security 層級阻擋。結論：**圖片必須觀察者手動上傳或改走 API route**。

### Finding 3: X Premium 25,000 字元限制

taiwandotmd 帳號有藍勾勾（Premium），推文上限 25,000 字元而非 280。Spore 全文（~150-300 字 + URL）輕鬆一則搞定無需拆串。
