# Mini Taiwan Pulse — 台湾交通リアルタイム3Dビジュアライゼーション 🌐

> 📖 **詳細版記事**：このリソースはシビックテックの深度研究記事にアップグレードされました。全文は [Mini Taiwan Pulse：一人のデータアナリストが台湾の交通の鼓動を呼吸する3D光の軌跡にした方法](/technology/mini-taiwan-pulse)（2026-04-19）をご覧ください。本ページはリソース一覧の索引エントリとして維持されています。

> **30秒でわかる概要：** 台湾の交通運輸のリアルタイム動態を3Dの光球と光の軌跡に変換するオープンソースプロジェクトです。フライトが空に弧を描き、船舶が海面に尾を引き、列車が軌道を駆け抜ける——23の切り替え可能なレイヤーで、台湾の「脈拍」を「見る」ことができます。

## なぜ注目すべきか

多くの人が台湾の地図を見ると、静的な輪郭しか見えません。Mini Taiwan Pulse が見せてくれるのは、**今まさに呼吸している島**です。

このプロジェクトの野心的な点は、各政府機関に散在するオープンデータ——フライト、船舶AIS、台湾鉄路・高速鉄路の時刻表、都市鉄道（MRT）路線、人口統計、気象観測——をすべて一枚の3D地図に統合することです。単なる点のマーカーではなく、光球、光の軌跡、彗星の尾のような視覚言語で、データが動く風景に変わります。

> **📝 キュレーターメモ**
> 台湾のオープンデータ基盤はアジアでもトップクラスです（[グローバル・オープンデータ・インデックス](https://index.okfn.org/)で複数回トップ10入り）。「データが公開されている」ことと「データが見られる」ことの間には大きなギャップが存在します。Mini Taiwan Pulse はその溝を埋めようとしています。

## 三つの鼓動

### 空 — フライトの光の軌跡 ✈️

台湾全土の14空港、1,500便以上のフライトのリアルタイム動態をカバー。各機体は発光する球体として表示され、背後に彗星のようなグラデーションの光の軌跡を残します。高度誇張倍率は調整可能（1倍〜5倍）で、低空路線と高空路線の違いが一目でわかります。

データソース：FlightRadar24 API。

### 海 — 船舶追跡 🚢

台湾周辺海域の船舶位置を青い光球で表示し、各船は30分間の尾跡を残します。システムはGPSの異常ジャンプや無効なMMSIを自動的にフィルタリングし、表示されるすべての光点が実在する船舶であることを保証します。

データソース：AIS（自動識別システム）船舶位置データ。

### 大地 — 六大軌道システム 🚄

おそらく最も印象的な部分です。6つの軌道システムが同時に動作します：

| システム                 | 規模                            |
| ------------------------ | ------------------------------- |
| 台湾鉄路（TRA）          | 265路線、333列車、車種別6色分類 |
| 台湾高速鉄路（THSR）     | 南北本線 + 支線                 |
| 台北MRT（TRTC）          | 8路線                           |
| 高雄MRT（KRTC）          | 紅線 + 橘線                     |
| 高雄ライトレール（KLRT） | 環状ライトレール                |
| 台中MRT（TMRT）          | 緑線 + 青線                     |

台湾鉄路の処理は特に複雑で、OD軌道マッチングや彰化三角線のような分岐路線には専用のエンジンが対応しています。

データソース：公開時刻表 + [OpenStreetMap](https://www.openstreetmap.org/) 軌道データ。

## 交通だけではない

移動する交通機関に加え、プロジェクトには複数の静的・分析レイヤーが重ねられています：

- **インフラ**：14空港の境界、535駅の光柱（高さ = 停車回数）、36基の灯台の3D回転ビーム
- **道路ネットワーク**：国道（赤）、省道（オレンジ）、自転車専用道（緑）、ズームに応じた幅の自動調整
- **人口分析**：H3六角グリッド人口ヒートマップ、昼夜の人流切り替え対応、9項目の人口指標
- **気象**：観測所リアルタイムデータ + 温度波3D曲面（0.03°格子解像度）
- **ニュース**：CNA中央通信社RSS + Gemini APIによるジオコーディング、ニュースイベントを地図上に表示
- **国道渋滞**：リアルタイムの渋滞度カラーコーディング

合計**23の独立切り替え可能なレイヤー**、10カテゴリ。

## 技術的な見どころ

- **TypeScript + Mapbox GL + Three.js**：2D地図はMapboxネイティブレンダリング、3D要素（光球、光の軌跡、光柱、温度曲面）はThree.jsでオーバーレイ
- **パフォーマンス最適化**：船舶はInstancedMeshでバッチレンダリング、ビューポートカリングで見えないオブジェクトのレンダリングを回避
- **色彩科学**：人口レイヤーにはPlasma / Viridis / Infernoなどの知覚的に均一なカラースケールを採用、log1p + gamma正規化で重い裾の分布を処理、色覚多様性に配慮
- **MITライセンス**：完全オープンソース、フォークとコントリビューションを歓迎

> **📝 キュレーターメモ**
> 光の軌跡のオーバーレイにアディティブブレンディングを採用するのは賢い選択です——複数の航路が重なる領域が自然に明るくなり、追加の統計グラフなしに航路の混雑度を視覚的に把握できます。

## オープンデータエコシステム

このプロジェクトが接続するデータソース自体が、台湾のオープンデータのガイドリストとなっています：

| データ                   | ソース                                             |
| ------------------------ | -------------------------------------------------- |
| フライトリアルタイム位置 | FlightRadar24 API                                  |
| 船舶AIS                  | 国際船舶自動識別システム                           |
| 鉄道時刻                 | 公開時刻表 + OSM                                   |
| バス/乗合バス/自転車     | [TDX公共交通データ](https://tdx.transportdata.tw/) |
| 人口統計                 | [SEGIS統計地理情報](https://segis.moi.gov.tw/)     |
| 気象観測                 | [中央気象署](https://www.cwa.gov.tw/)              |
| 洋上風力発電所           | 経済部エネルギー局                                 |
| ニュースイベント         | 中央通信社CNA RSS                                  |
| 空港/港湾/駅の境界       | [OSM Overpass API](https://overpass-turbo.eu/)     |

⚠️ **注目すべき点：** 台湾の[TDX運輸資料流通服務](https://tdx.transportdata.tw/)は、全国の公共交通データを統一的に標準化した数少ない政府プラットフォームの一つで、バス、乗合バス、鉄道、自転車などをカバーし、APIドキュメントが整備され無料で利用できます。これは世界的に見ても珍しい取り組みです。
