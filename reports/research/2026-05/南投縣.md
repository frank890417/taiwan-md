# Research Report: 南投縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：日月潭／921 集集大地震（1999/09/21 01:47:15.9，規模 7.3，深 8km，震央集集鎮距日月潭西偏南 9.2km）／中興新村（省政府舊址）／合歡山公路 + 武嶺／清境農場／溪頭妖怪村／凍頂烏龍茶（鹿谷）／阿里山高山茶（信義部分）／集集綠色隧道／埔里酒廠 + 紹興酒／霧社事件 1930／玉山國家公園（部分跨南投信義鄉）
- **世代差異**：
  - 戰前老世代知「霧社事件 1930」（莫那魯道領導賽德克族抗日 + 後續第二次霧社事件）
  - 戰後 50-70s 世代知「中興新村 1957」（台灣省政府臨時辦公處 → 1998 精省）
  - 80-90s 世代知「凍頂烏龍 + 阿里山高山茶」（南投山區茶葉產業興起）
  - 千禧後世代知「921 地震 1999」（震央就在集集，南投縣全境重災） +「合歡山騎登」+「日月潭環湖賽」
- **外人 vs 在地人**：外人覺得「日月潭遊船 + 清境綿羊秀」就走人；在地人知「我們是唯一不靠海的縣」「921 震央在集集，我們花 20 多年才復原一半」「霧社是賽德克族的傷口不是觀光景點」「中興新村還有省政府時代的椰子樹但功能已經被精省抽空」

### 多元面貌

- **主流敘事**：「日月潭 + 茶葉的觀光天堂」（過度浪漫化）vs 「921 災區」（過度悲觀）
- **支線敘事**：
  - **全台唯一不靠海的縣**：13 鄉鎮市，從西部彰化邊界到東部花蓮邊界，海拔從 100m 到 3,952m（玉山，含跨南投信義鄉部分）
  - **921 集集大地震**：1999/09/21 01:47:15.9，規模 7.3，深 8km，震央集集鎮，全台死亡 2,415 + 失蹤 29 + 受傷 11,305 + 全倒 51,711 棟 + 半倒 53,768 棟。南投縣受創最鉅，埔里、竹山、名間、中寮、國姓、草屯重災
  - **霧社事件 1930/10/27**：賽德克族莫那魯道領導抗日 + 後續第二次霧社事件（保護蕃收容所被親日蕃襲擊）
  - **中興新村 1957**：台灣省政府臨時辦公處（避免兩岸戰爭波及台北）→ 1998 精省後功能抽空 + 至今椰子樹大道猶存
  - **3 個原住民族同住**：賽德克族（仁愛鄉，2008 從泰雅族分出）+ 布農族（信義鄉，巒社、丹社、卡社等）+ 鄒族（信義鄉部分）+ 邵族（日月潭周邊，2001 正名第 10 族）
  - **4 種高山茶**：凍頂烏龍（鹿谷）+ 阿里山高山茶（信義一部分）+ 杉林溪茶（鹿谷竹山）+ 廬山茶（仁愛）
  - **八堡圳起點**（名間鄉）：康熙年間 1709 施世榜建，全台最古老水利系統之一
  - **2008 邵族正名第 10 族**：日月潭水沙連邵族 2001 從平埔族正名
- **fault lines**：
  - 觀光（日月潭 + 清境）vs 災後重建（921 多重災區還在恢復）
  - 賽德克 + 布農 + 邵族 vs 漢人（中興新村漢人 vs 山區原住民）
  - 1930 霧社事件 vs 今天的霧社觀光化（賽德克族傷口跟觀光商業）
  - 中興新村精省後的尷尬（功能抽空 vs 椰林大道猶存）
  - 茶葉文化 vs 環境（高山茶開發 vs 水土保持）

### 想法跟感受

- **正面**：南投人對「唯一不靠海」「日月潭是台灣的中心」有強烈認同 / 對賽德克族 + 布農族 + 邵族 + 漢人多元文化的自信 / 對中興新村椰林大道 + 凍頂烏龍 + 集集綠色隧道的在地驕傲
- **負面**：「921 災區」標籤的疲乏 / 中興新村精省後失落 / 山區人口外流 / 阿里山高山茶被外縣搶名（一部分屬南投信義但通稱阿里山）
- **矛盾感**：唯一不靠海卻是台灣最早地震頻發 / 中興新村是省政府卻精省後變遺跡 / 霧社事件 1930 + 921 1999 + 廬山溫泉遷村 2015 — 三次集體創傷

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 賽德克族 + 布農族 + 鄒族 + 邵族（日月潭）千年居住
  - 17 世紀後漢人從西部進入南投平原
  - 1709 八堡圳建立（康熙 48 年，施世榜，台灣最早水利系統之一）
- **關鍵轉折**：
  - 1723 雍正元年彰化縣設立（含今南投部分）→ 1875 設南投縣
  - 1895 日治：南投廳 + 1920 改台中州
  - 1930/10/27 霧社事件（莫那魯道領導賽德克族抗日 + 第二次霧社事件）
  - 1945 戰後：南投縣設立
  - 1957 台灣省政府臨時辦公處設中興新村
  - 1998/12 精省（省政府虛級化）→ 中興新村功能抽空
  - 1999/09/21 集集大地震（規模 7.3，全台死亡 2,415，南投全境重災）
  - 2001 邵族正名第 10 個原住民族
  - 2008 賽德克族正名第 14 族（從泰雅族分出）
  - 2015 廬山溫泉遷村（921 + 後續颱風造成基地破壞）
- **當代意義**：南投是台灣「唯一不靠海 + 多重歷史創傷 + 多元族群」的典範。921 地震 + 霧社事件 + 中興新村精省 + 廬山溫泉遷村 — 四個故事撐起南投的當代意義

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 南投是台灣「唯一不靠海」的物理座標（西部彰化邊界到東部花蓮邊界 + 玉山 3,952m）
- 921 地震 1999 是台灣戰後傷亡最嚴重的自然災害（2,415 死 + 全倒 51,711 棟）
- 霧社事件 1930 是台灣原住民抗日最具代表性的事件（莫那魯道 + 賽德克族）
- 中興新村 1957-1998 是台灣「省政府時代」的物理見證（精省 26 年後椰林大道猶存）
- 邵族 2001 + 賽德克族 2008 兩次正名，南投正名運動最深
- 對讀者：「為什麼台灣唯一不靠海的縣，是地震 + 抗日 + 行政變革 + 多元族群的集合點？」這個 frame 讓人重新看見「中心 vs 邊陲」的物理意義

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：中央山脈 + 玉山山脈 + 阿里山山脈 + 雪山山脈南段 / 集水區為濁水溪上游、烏溪、北港溪 + 八堡圳 / 海拔 100m（南投市區）到 3,952m（玉山）/ 13 鄉鎮市
- **誰在這裡生活**：賽德克族（仁愛鄉，1930 霧社事件後遷移整合）+ 布農族（信義鄉，卓社 + 卡社 + 巒社 + 丹社 + 郡社 5 群）+ 邵族（日月潭，2001 正名第 10 族）+ 鄒族（信義鄉部分）+ 漢人閩客（清領拓墾後主流）
- **人和地有什麼共生**：山地 → 高山茶（凍頂 + 杉林溪 + 廬山 + 阿里山部分）/ 集水區 → 日月潭發電（1934 武界壩 + 大觀電廠）+ 八堡圳灌溉 / 中興新村 → 省政府時代的椰林大道 + 精省後遺跡 / 玉山 → 玉山國家公園 + 布農族傳統 / 霧社 → 賽德克族傷口 + 觀光化

### 切入點清單（候選 hooks）

1. **唯一不靠海的縣 + 921 震央在集集**（1999/09/21 規模 7.3 + 距日月潭 9.2km + 全台 2,415 死 + 南投受創最鉅 + 26 年後仍部分重建中）

3. **中興新村 1957-1998 省政府時代**（49 萬人精省前 → 精省後功能抽空 + 椰林大道猶存）
4. **3 次正名 + 多元族群**（邵族 2001 第 10 族 + 賽德克族 2008 第 14 族 + 布農族傳統八部合音）
