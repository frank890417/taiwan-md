# Research Report: 台東縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：南迴公路／知本溫泉／池上米 + 池上便當／太麻里金針山／綠島（人權紀念園區，火燒島）／蘭嶼（達悟族，核廢料儲存場 1982）／卑南遺址（史前文化博物館 2002）／陳建年（卑南族金曲歌王）／紅葉少棒 1968／都蘭山／東河鄉
- **世代差異**：
  - 戰前老世代知「卑南」（卑南社 + 卑南覓 + 卑南族傳統八社）+ 1875 沈葆楨開山撫番 + 八通關古道
  - 戰後 50-70s 世代知「火燒島」（綠島政治犯監獄 1951-1987）+「南橫公路 1972 通車」+「紅葉少棒 1968」
  - 80-90s 世代知「金曲獎台東幫」（陳建年 1999 金曲歌王 + 紀曉君 + 巴奈 + 胡德夫等）+「池上稻浪」
  - 千禧後世代知「蘭嶼核廢料」（1982 起 + 1988 反核廢驅逐惡靈運動 + 2019 蔡英文 25.5 億補償）+「綠島人權公園」
- **外人 vs 在地人**：外人覺得「池上 + 蘭嶼 + 知本溫泉」就走人；在地人知「21 萬人散在 3,515 km² 全國第三大縣，密度是台北的 1/100」「蘭嶼天空 6 個達悟族部落 + 核廢料 42 年沒移走」「紅葉村今天 30 戶，但 1968 那 7 個孩子改變了台灣棒球史」「卑南遺址 5300-3500 年前比台灣本島任何漢人聚落都早」

### 多元面貌

- **主流敘事**：「東部最後一塊淨土」+「離島南島民族」（過度浪漫化）vs 「最偏遠縣份」（過度悲觀）
- **支線敘事**：
  - **21 萬人散在 3,515 km²**：全國第三大縣（次於花蓮、南投），密度約 60 人/km²，是台北市的 1/100
  - **6 個原住民族同住**：阿美族（東海岸長濱、東河、成功）+ 卑南族（南王、知本、初鹿 8 社）+ 排灣族（達仁、金峰、太麻里）+ 魯凱族（部分達仁鄉）+ 布農族（海端鄉，1934 從南投遷來）+ 達悟族（蘭嶼鄉 6 部落）
  - **蘭嶼核廢料 1982**：1975 蔣經國同意設貯存場 + 1982 第一批 10,008 桶 + 1988/02/20 達悟族首次反核廢驅逐惡靈運動 + 累積 97,672 桶低放射性廢料 + 1996 投石封港擋運送 + 2019/11 蔡英文 25.5 億補償
  - **綠島火燒島 1951-1987**：監獄 + 政治犯 + 1992 解嚴後改人權紀念園區 + 朝日溫泉（海底溫泉）+ 浮潛
  - **卑南遺址 5300-3500 年前**：史前文化博物館 2002 開館 + 玉器 + 石棺 + 比台灣本島任何漢人聚落都早 1,000+ 年
  - **紅葉少棒 1968**：紅葉村 7 個布農族孩子 + 1968/08/25 4:0 擊敗日本和歌山隊（後證實是日本商業隊非國家隊）+ 改變台灣棒球史
  - **陳建年金曲歌王 1999**：卑南族 + 海洋系列專輯 + 警察身分 + 台東代表音樂人
  - **池上米 + 池上便當**：池上鄉 + 中央山脈 + 海岸山脈雙山脈下的縱谷米倉
  - **太麻里金針山 + 知本溫泉**：太麻里日出觀光 + 知本碳酸氫鈉泉
  - **南迴公路 + 南迴鐵路電氣化**：1985 南迴公路通車 + 2020 南迴鐵路電氣化通車
- **fault lines**：
  - 蘭嶼達悟族 vs 台電核廢料的 42 年抗爭
  - 6 族同住 + 漢人少數的特殊族群結構（原住民約 35%，全台最高比例）
  - 21 萬人散在 3,515 km² 的低密度跟交通不便
  - 綠島火燒島 + 蘭嶼核廢料：兩個離島承擔台灣現代化的陰影
  - 池上稻作 vs 太麻里日出 vs 蘭嶼海洋的三種觀光地景
  - 紅葉少棒（榮光）vs 紅葉村今天（30 戶）的對比

### 想法跟感受

- **正面**：台東人對「最後淨土」「東部慢活」有強烈認同 / 對 6 族同住 + 史前文化 + 池上米的文化驕傲 / 對綠島人權園區 + 蘭嶼反核 42 年的政治意識
- **負面**：「偏遠縣份」「人口外流」的失落 / 蘭嶼核廢料 42 年沒移走的不公 / 紅葉少棒榮光跟今天紅葉村 30 戶的對比
- **矛盾感**：21 萬人散在 3,515 km² 的低密度 vs 卑南遺址 5300 年前比西部漢人聚落都早 / 蘭嶼天空 6 個部落 + 核廢料同住 / 紅葉少棒 1968 改變台灣棒球史 vs 紅葉村今天 30 戶

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 卑南族 + 阿美族 + 排灣族 + 魯凱族 + 達悟族（蘭嶼）千年居住
  - 卑南文化 5300-3500 年前（卑南遺址）
  - 17 世紀後西班牙、荷蘭未深入東海岸
- **關鍵轉折**：
  - 1875 沈葆楨開山撫番 + 八通關古道（玉里經太麻里到台東）
  - 1895 日治：台東廳 + 1934 布農族從南投遷入海端鄉
  - 1945 戰後：台東縣設立
  - 1951 綠島成為政治犯監獄（火燒島）
  - 1968/08/25 紅葉少棒擊敗日本和歌山隊 4:0
  - 1972 南橫公路通車（玉山 → 海端 → 台東）
  - 1975 蔣經國同意蘭嶼設核廢料貯存場
  - 1982 第一批 10,008 桶核廢料進入蘭嶼
  - 1985 南迴公路通車（楓港 → 台東）
  - 1987 解嚴 + 1992 綠島政治犯監獄改人權紀念園區
  - 1988/02/20 達悟族首次反核廢驅逐惡靈運動
  - 1996 蘭嶼人投石封港擋運送
  - 1999 陳建年金曲歌王
  - 2002 史前文化博物館開館 + 卑南遺址公開
  - 2019/11 蔡英文 25.5 億蘭嶼核廢料補償
  - 2020 南迴鐵路電氣化全線通車
- **當代意義**：台東縣是台灣「最後淨土」+「6 族同住」+「離島承擔現代化陰影」的縮影。21 萬人散在 3,515 km² 的低密度，蘭嶼核廢料 42 年抗爭，綠島人權園區 — 三個故事撐起台東的當代意義

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 台東縣是台灣「最低密度 + 最高原住民比例」的物理座標（21 萬 / 3,515 km² / 原住民約 35%）
- 卑南遺址 5300-3500 年前是台灣最早的人類聚落遺址之一（比西部漢人 1684 諸羅縣早 5000+ 年）
- 蘭嶼核廢料 42 年抗爭是台灣「環境正義 + 原住民權利」最具代表性的案例
- 綠島火燒島 1951-1987 是台灣戒嚴時期政治壓迫的物理見證
- 紅葉少棒 1968 是台灣棒球史的起點（雖然事後證實對手是日本商業隊）
- 對讀者：「為什麼台灣最早的人類聚落 + 最深的環境正義抗爭 + 最低人口密度都在同一個縣？」這個 frame 拆開後讓人重新看見「邊緣」的歷史厚度

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：花東縱谷（中央山脈 + 海岸山脈夾出來的稻作平原）+ 海岸山脈太平洋面 + 中央山脈東面 + 大武山以南山地 + 綠島（火山岩）+ 蘭嶼（火山岩）+ 卑南溪沖積平原 + 太麻里溪
- **誰在這裡生活**：卑南族（南王、知本、初鹿，臺東市區）+ 阿美族（東海岸長濱、東河、成功，海岸 + 縱谷北段）+ 排灣族（達仁、金峰、太麻里南部）+ 魯凱族（部分達仁鄉）+ 布農族（海端鄉，1934 從南投遷來）+ 達悟族（蘭嶼鄉 6 部落）+ 漢人閩南、客家少數
- **人和地有什麼共生**：縱谷 → 池上米 + 富里米 + 關山米 / 海岸山脈 → 阿美族傳統 + 太平洋 / 中央山脈 → 卑南族 + 布農族 / 蘭嶼 → 達悟族 + 飛魚祭 + 核廢料 / 綠島 → 火燒島記憶 + 朝日溫泉 / 太麻里 → 排灣族 + 金針山日出
