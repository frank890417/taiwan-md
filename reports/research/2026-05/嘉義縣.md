# Research Report: 嘉義縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：阿里山小火車（高山鐵路）／阿里山雲海日出／鄒族部落（達邦、特富野、來吉）／故宮南院（2015/12/28 開館，太保市故宮大道 888 號）／太保高鐵嘉義站／民雄鬼屋／東石蚵仔／布袋鹽田／朴子市區
- **世代差異**：
  - 戰前老世代知「鄒族與阿里山」（1900 鳥居龍藏記錄 + 1908 阿里山鐵路規劃 + 1912 通車的森林鐵道）
  - 戰後 50-70s 世代知「林業 + 阿里山木材」（戰後檜木輸出顛峰 + 1989 林業政策轉向）
  - 80-90s 世代知「縣市分治 + 阿里山觀光」（1950 嘉義縣市分治 + 1995 縣治從嘉義市遷至太保）
  - 千禧後世代知「故宮南院 + 高鐵 BRT」（2015 故宮南院 + 高鐵嘉義站太保特區 + BRT 連嘉義市）
- **外人 vs 在地人**：外人覺得「嘉義縣 = 阿里山」就走人；在地人知「我們縣治在太保不是嘉義市」「故宮南院蓋了 15 年才開」「鄒族在達邦、特富野，蘭潭水庫的水是給嘉義市的」「東石、布袋的蚵田跟太保高鐵站是兩個世界」

### 多元面貌

- **主流敘事**：「阿里山就是嘉義縣」（過度單一化）vs 「被嘉義市切割的甜甜圈」（行政地理 frame）
- **支線敘事**：
  - 嘉義縣 49 萬人圍繞著被切割出去的嘉義市 27 萬人 — 兩個嘉義的不對稱
  - 1786 林爽文之役（諸羅城被圍兩個月 → 乾隆改名「嘉義」，那年的諸羅縣城在今嘉義市區，嘉義縣是後來才出現的）
  - 1950 嘉義縣市分治 → 1982 嘉義市升格省轄市 → 1991 縣治從嘉義市遷至太保
  - 鄒族（阿里山鄉達邦、特富野；信義鄉部分）+ 高一生（1908-1954，鄒族長老，白色恐怖犧牲者）
  - 故宮南院 2003 行政院決定 → 2015/12/28 開館（風波 + 延宕 15 年才開）
  - 高鐵嘉義站太保特區 + BRT 連嘉義市 + 故宮南院形成「太保新核心」嘗試
  - 1908 阿里山鐵路規劃 + 1912 嘉義至二萬坪 66.6km 通車（屬嘉義市起點，但林業集散地往嘉義縣山區延伸）
  - 西部沿海：東石、布袋、朴子（蚵田、鹽田、漁港）vs 東部山區：阿里山、番路、竹崎、梅山（茶葉、檜木、原住民）
  - 18 鄉鎮市
- **fault lines**：
  - 嘉義縣 vs 嘉義市（行政切割 1950 / 縣治遷太保 1991）
  - 西部沿海漁業（東石、布袋） vs 東部山區林業觀光（阿里山）
  - 鄒族部落（山區）vs 漢人移民（平地） + 高一生白色恐怖犧牲歷史
  - 太保新核心 vs 嘉義市傳統核心（兩個中心拉扯）
  - 故宮南院的「文化政治」（藍綠看法分歧）

### 想法跟感受

- **正面**：嘉義縣人對「阿里山是我們的」有強烈認同 / 對東石蚵仔 + 布袋鹽田 + 朴子市區三個西部沿海節點的自信 / 對故宮南院（雖然蓋了 15 年）開館後的觀光效益的期待
- **負面**：被嘉義市切割的失落（縣治遷太保的尷尬 + 沒有自己的「市」感覺）/ 西部沿海跟東部山區資源分配不均 / 人口外流到嘉義市跟外縣
- **矛盾感**：阿里山小火車 (1912) + 故宮南院 (2015) + 高鐵嘉義站 (2007) 三個時代的觀光策略疊在同一個縣 / 縣治在太保但全國認知「嘉義就是嘉義市」

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 鄒族（阿里山山區） + 平埔族洪雅族（諸羅山社、阿里山社平地一帶） + 1621 顏思齊登陸笨港（屬今嘉義縣北港 / 新港一帶，不是嘉義市）
  - 1684 諸羅縣設立（清領初期，今嘉義縣 + 嘉義市 + 雲林部分）
  - 1786 林爽文之役諸羅城守城（在今嘉義市），1787 乾隆改名「嘉義」（嘉義縣的縣名也由此來）
- **關鍵轉折**：
  - 1895 日治：嘉義廳 → 1920 嘉義街升市 → 嘉義縣治留嘉義市
  - 1908 阿里山鐵路規劃 → 1912 通車（66.6km 至二萬坪）+ 1914 延伸阿里山
  - 1950 嘉義縣市分治：嘉義市獨立省轄市，嘉義縣治留嘉義市（暫）
  - 1982 嘉義市升格為省轄市（位階提高）
  - 1991 嘉義縣治從嘉義市遷至太保市
  - 2003 行政院決定故宮南院設嘉義太保市
  - 2007 高鐵嘉義站太保特區啟用 + BRT 連嘉義市
  - 2015/12/28 故宮南院開館（規劃到落成 15 年風波延宕）
  - 1947 二二八嘉義事件部分波及阿里山（高一生 1954 白色恐怖犧牲）
- **當代意義**：嘉義縣是台灣「行政切割後的圍繞城」典範 — 縣治在太保，但全國認知還是「嘉義就是嘉義市」。故宮南院 + 高鐵嘉義站試圖建立「太保新核心」，跟阿里山小火車的傳統觀光形成三個時代的觀光策略並存

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 嘉義縣是台灣「行政切割」最戲劇性的案例（49 萬縣 vs 27 萬市 / 縣治遷太保 31 年）
- 阿里山小火車是台灣高山鐵道的核心遺產（1912 通車 + 戰後檜木輸出顛峰 + 1989 林業政策轉向 + 觀光化）
- 鄒族高一生（1908-1954）是台灣原住民現代化跟白色恐怖犧牲的縮影
- 故宮南院 + 高鐵嘉義站 + BRT 是台灣「2000s 後新城市核心嘗試」的典型 case（成功 vs 失敗的判斷各有看法）
- 對讀者：「為什麼一個 49 萬人的縣，要把縣治放在 4 萬人的太保？」這個 frame 拆開後讓人重新看見「行政地理 vs 文化認知」的張力

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：嘉南平原東緣（西部沿海：東石、布袋、朴子 - 鹽分地帶 + 漁港）→ 嘉南平原中段（民雄、新港、太保 - 稻作 + 高鐵特區）→ 阿里山山脈西麓（番路、竹崎、梅山、阿里山 - 茶葉 + 檜木 + 鄒族）
- **誰在這裡生活**：鄒族（阿里山山區 - 達邦、特富野、來吉）+ 平埔族洪雅族（諸羅山社、阿里山社平地一帶，已漢化）+ 17 世紀漳泉移民 + 日治日本人（林業 + 行政）+ 戰後外省移民
- **人和地有什麼共生**：阿里山林業 → 嘉義縣作為木材集散地 + 觀光化 / 嘉南平原 → 米倉 + 高鐵特區 / 西部沿海 → 蚵田（東石、布袋）+ 鹽田（布袋）+ 漁港 / 山區 → 阿里山茶葉 + 鄒族傳統 / 故宮南院 → 太保新核心嘗試

### 切入點清單（候選 hooks）

1. **三個時代的觀光策略疊在同一個縣**（1912 阿里山小火車傳統 + 2007 高鐵嘉義站 + 2015 故宮南院 → 三層觀光地景並存）
2. **49 萬縣 vs 27 萬市的甜甜圈**（嘉義縣圍繞嘉義市，1950 分治 + 1991 縣治遷太保 + 全國認知斷裂）
3. **故宮南院蓋了 15 年才開**（2003 行政院決定 + 2015 開館，中間風波 + 延宕 + 預算爭議 + 完工典禮在總統大選前）
4. **鄒族高一生 1908-1954**（鄒族長老 + 日治時期接受教育 + 1947 二二八嘉義事件協助維持秩序 + 1954 白色恐怖被槍決）
5. **阿里山小火車的三段歷史**（1912 通車運木材 → 1989 林業政策轉向觀光鐵路 → 2009 八八風災部分坍方 → 至今仍部分復原中）
6. **東石蚵田 vs 太保高鐵特區**（西部沿海蚵農 vs 太保新核心嘗試 → 一個縣兩個世界）
7. **1786 諸羅城圍城 + 乾隆改名 1787**（縣名「嘉義」起源 → 雖然事件在今嘉義市，但「嘉義縣」也由此來，跟嘉義市共享這段歷史）

### 預期核心矛盾候選（待 Stage 1.4 收斂為 ≤ 30 字）

- A：三個時代的觀光策略疊在同一個縣，但沒有一個替「嘉義縣」本身定位
