# Research Report: 屏東縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：墾丁／鵝鑾鼻燈塔／龍磐草原／恆春古城／屏東萬丹紅豆／東港黑鮪魚／潮州燒冷冰／萬巒豬腳／六堆客家文化／排灣族（霧台、瑪家、來義、牡丹）／魯凱族（霧台鄉）／琉球嶼（小琉球）／墾丁春吶
- **世代差異**：
  - 戰前老世代知「阿緱聽」（屏東舊名，日治設廳）+「牡丹社事件 1874」（日軍登陸 + 排灣族抗戰）
  - 戰後 50-80s 世代知「屏東糖廠 + 屏東農業」（甘蔗 + 蓮霧 + 檳榔 + 椰子）
  - 90s 世代知「墾丁觀光」（白沙灣 + 大灣 + 後壁湖 + 1990s 春吶起源）
  - 千禧後世代知「88 風災」（2009/8/8 莫拉克 + 林邊佳冬泡水 + 排灣族部落重建）
- **外人 vs 在地人**：外人覺得「墾丁 + 黑鮪魚」就走人；在地人知「我們有 5 個原住民族 + 客家六堆 + 閩南三大族群」「88 風災時林邊整個泡在海水裡」「恆春半島冬天的落山風像沒人開過冷氣的吹風機」「牡丹社事件不只是 1874，今天牡丹鄉還有 stories」

### 多元面貌

- **主流敘事**：「南國度假天堂」+「黑鮪魚故鄉」（觀光化過度浪漫）vs 「南部偏鄉」（過度悲觀）
- **支線敘事**：
  - **5 個原住民族同住**：排灣族（北排灣霧台、瑪家、泰武 + 南排灣春日、獅子、牡丹）+ 魯凱族（霧台鄉，跟北排灣鄰住）+ 平埔族馬卡道（屏東平原已漢化）+ 卑南族（小部分）+ 阿美族（極少）
  - **客家六堆**：內埔、竹田、萬巒、麟洛、長治、新埤 6 鄉鎮（清代義民組織延續至今的客家庄文化區，最早可追溯 1721 朱一貴之亂）
  - **三族群混居比例**：閩南 70% / 客家 12% / 原住民 8% / 外省 10%（粗估）
  - **1874 牡丹社事件**：日軍登陸社寮 5/8 + 5/22 排灣族石門埋伏 + 阿祿古父子身亡 + 改變台灣國際地位 + 清廷積極治理
  - **1871 琉球漂民事件**：54 名琉球人在八瑤灣遭排灣族殺害 + 12 名楊友旺護送離開
  - **88 風災 2009/8/8**：林邊鄉、佳冬鄉被林邊溪潰堤淹沒 + 太武村小林村高雄但屏東山區排灣族部落（瑪家、霧台、泰武）也重災
  - **恆春古城 1875**：劉永福 + 沈葆楨開山撫番後設恆春縣城
  - **萬金聖母聖殿 1861**：西班牙傳教士設立，是台灣第一座天主教堂
  - **小琉球（琉球嶼）**：唯一獨立離島鄉，珊瑚礁地形 + 潮間帶生態 + 觀光
  - **林邊蓮霧、屏東萬丹紅豆、潮州燒冷冰、東港櫻花蝦、萬巒豬腳、客家粄條**：飲食地景
- **fault lines**：
  - 觀光（墾丁）vs 農漁業（屏東平原 + 沿海）vs 原住民部落（山區）
  - 1874 牡丹社事件的歷史敘事：日軍「報復」vs 排灣族「保衛」vs 琉球「漂民」三重視角
  - 88 風災重建（2009-）vs 在地產業轉型（蓮霧 vs 海水倒灌的後續）
  - 六堆客家 vs 閩南 vs 排灣族 vs 魯凱族的文化交融與張力
  - 恆春半島冬天觀光淡季 vs 落山風生態

### 想法跟感受

- **正面**：屏東人對「最南端」「多元族群」有強烈認同 / 對墾丁、黑鮪魚、萬丹紅豆、萬巒豬腳、潮州燒冷冰的在地驕傲 / 對六堆客家 235 年歷史的文化延續感
- **負面**：「南部框架」的失落（人口外流 + 觀光淡季長 + 大學少）/ 88 風災重建的長期成本 / 排灣族部落跟主流敘事的距離
- **矛盾感**：5 族同住的多元 vs 主流敘事只記得墾丁 / 1874 牡丹社事件改變了台灣，但牡丹今天仍在發生（小規模事件持續）/ 觀光繁榮 vs 農業沒落 / 蓮霧成名跟 88 風災的雙刃

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 排灣族 + 魯凱族 + 馬卡道族（平埔）千年居住
  - 1664 鄭氏跟荷蘭人撤離後屏東平原開拓
  - 1721 朱一貴之亂後客家六堆組織興起（內埔、竹田、萬巒、麟洛、長治、新埤）
  - 1721 後閩粵移民移居屏東平原
- **關鍵轉折**：
  - 1861 萬金聖母聖殿建立（西班牙傳教士，台灣首座天主堂）
  - 1871 琉球漂民事件（54 名遭殺害）
  - 1874 牡丹社事件（日軍登陸 5/8 + 5/22 戰鬥 + 阿祿古父子身亡）→ 改變台灣國際地位
  - 1875 沈葆楨開山撫番 + 設恆春縣
  - 1895 日治：阿緱聽 + 1920 改高雄州屏東郡
  - 1945 戰後：屏東縣設立（從高雄州劃出）
  - 1980s 屏東糖廠停產 + 觀光業興起
  - 1980 第一屆春吶（墾丁春浪音樂節雛形）
  - 2009/8/8 莫拉克颱風 88 風災（林邊 + 佳冬 + 山區排灣族部落重災）
  - 2010s 排灣族正名運動深化 + 牡丹社事件 150 週年（2024）紀念
- **當代意義**：屏東縣是台灣「最南端 + 最多元族群」的典範 — 5 個原住民族 + 客家六堆 + 閩南 + 外省四大族群同住 79 萬人。1874 牡丹社事件改變了台灣國際地位，2009 八八風災重塑了沿海跟山區的生活方式

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 屏東是台灣「最南」的物理座標（鵝鑾鼻北緯 21°54′）+ 1874 牡丹社事件改變台灣國際地位的歷史轉折點
- 5 個原住民族 + 客家六堆 + 閩南 + 外省同住，「南方多元」是物理現實不是政治口號
- 88 風災是台灣「氣候變遷 + 海岸線退縮」最具代表性的災難案例（林邊海水倒灌 + 山區部落滅村）
- 萬金聖母聖殿 1861 是台灣第一座天主教堂 — 比馬偕（淡水牛津學堂 1882）早 21 年的基督宗教遺產
- 對讀者：「為什麼一個 79 萬人的縣，需要 5 個原住民族 + 客家六堆 + 三大族群同時存在？」這個 frame 拆開後讓人看見「多元」的物理代價跟禮物

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：屏東平原（高屏溪沖積扇）+ 中央山脈南端（北大武山 3,092m）+ 恆春半島（佳洛水 + 龍磐草原 + 鵝鑾鼻）+ 琉球嶼（珊瑚礁離島）/ 氣候：熱帶季風 + 冬季落山風 + 颱風路徑常經過
- **誰在這裡生活**：排灣族（北排灣霧台、瑪家、泰武 / 南排灣春日、獅子、牡丹）+ 魯凱族（霧台鄉，跟北排灣鄰住）+ 馬卡道族（屏東平原已漢化）+ 1721 後閩粵移民 + 日治日本人（糖廠 + 行政）+ 戰後外省（軍公教 + 國軍）
- **人和地有什麼共生**：屏東平原 → 蓮霧 + 萬丹紅豆 + 檳榔 + 椰子 + 蕉農 / 山區 → 排灣族 + 魯凱族傳統 + 山地觀光 / 恆春半島 → 墾丁觀光 + 落山風 / 沿海 → 東港黑鮪魚 + 林邊蓮霧（海水倒灌風險）+ 養殖漁業

### 切入點清單（候選 hooks）

1. **5 族同住的物理現實**（排灣 + 魯凱 + 客家六堆 + 閩南 + 外省，79 萬人 33 鄉鎮市，「南方多元」是地理結構不是口號）
2. **1874 牡丹社事件 vs 今天的牡丹**（日軍登陸 5/8 + 5/22 排灣族石門埋伏 + 阿祿古父子身亡 → 改變台灣國際地位 → 牡丹鄉今天 5,000 人持續守著祖先記憶）
3. **88 風災 2009/8/8 的雙重打擊**（林邊海水倒灌泡 1 個月 + 山區排灣族部落滅村 → 屏東兩種地理同時被打 + 重建 16 年）
4. **客家六堆 235 年延續**（1721 朱一貴之亂義民組織 → 內埔竹田萬巒麟洛長治新埤 6 鄉鎮 → 客家硬頸的南方版本，跟苗栗 + 新竹客家不同）
5. **萬金聖母聖殿 1861**（西班牙傳教士 + 台灣第一座天主教堂 + 比馬偕早 21 年 + 屏東萬金村萬巒鄉的小村莊裡）
6. **墾丁春吶 1995 起源**（屏東半島冬天落山風生態 + 觀光季 vs 淡季 + 春吶從地下音樂變主流觀光）
7. **東港黑鮪魚拍賣 + 鹽埔 + 大鵬灣**（東港的拍賣現場 vs 大鵬灣國家風景區 vs 鹽埔漁港的轉型）
