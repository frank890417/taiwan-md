# Research Report: 彰化縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：鹿港老街（「一府二鹿三艋舺」的二鹿）／鹿港龍山寺（1786 重建）／八卦山大佛（1961 竣工，22m，曾稱亞洲第一）／鹿港天后宮（媽祖信仰）／鹿港民俗博物館／員林雞腿便當／田尾公路花園／二水跑水節（八堡圳）／彰化扇形車庫（1922）／鹿港老街桂花巷／鹿港龍山寺
- **世代差異**：
  - 戰前老世代知「一府二鹿三艋舺」（鹿港 1717 始見地圖 + 1728 米倉 + 二鹿時代）+「1723 彰化縣設立」（雍正元年從諸羅縣分出，非 1812）
  - 戰後 50-70s 世代知「彰化扇形車庫 1922」（日治蒸氣火車迴轉中心 + 戰後保留至今）+ 員林糖廠
  - 80-90s 世代知「八卦山大佛 1961」（22m 高 + 亞洲第一大佛 + 彰化縣地標）+ 田尾公路花園
  - 千禧後世代知「員林市區擴張」（縣治彰化市 vs 員林市人口接近）+「鹿港觀光化」
- **外人 vs 在地人**：外人覺得「鹿港老街吃肉圓 + 八卦山看大佛」就走人；在地人知「鹿港的『艋舺』時代結束 200 年了，但鹿港人還在生活」「員林雞腿便當的車站文化跟北部不一樣」「我們是 124 萬人全台第三大縣（次於新北、台中）但全國只記得鹿港」「田尾公路花園是台灣花卉中心但被荷蘭花卉產業搶風采」

### 多元面貌

- **主流敘事**：「一府二鹿三艋舺的鹿港」（過度集中歷史）vs 「124 萬人的彰化平原」（過度抽象）
- **支線敘事**：
  - **彰化平原是台灣第二大平原**（次於嘉南平原），非台灣最大（Stage 0 預設誤）
  - **1723 設立彰化縣**（雍正元年從諸羅縣分出，含舊虎尾溪以北至大甲溪以南）→ 非 1812
  - **鹿港歷史**：1717 始見諸羅縣志地圖 + 1728 米倉 16 間 + 1788 龍山寺重建 + 「一府二鹿三艋舺」時代 18 世紀末-19 世紀初
  - **八卦山大佛 1961 竣工**：22m 高 + 6 層內部空間 + 戰後台灣最早大佛之一
  - **彰化扇形車庫 1922 建成**：日治蒸氣火車迴轉設備 + 戰後保留 + 1990s 解體爭議 + 2000s 文化資產保存
  - **八堡圳 1709 起點在二水**（康熙 48 年，施世榜 + 林先生傳說，跨南投彰化）+ 二水跑水節
  - **員林市 + 鹿港鎮 vs 彰化市**：縣治彰化市 24 萬 + 員林市 12 萬 + 鹿港鎮 8 萬，三個都市核心
  - **客家飛地田尾鄉 + 永靖鄉**（中部客家飛地）
  - **彰化縣 26 鄉鎮市**：1 縣轄市（彰化）+ 7 鎮（員林、鹿港、和美、北斗、二林、田中、溪湖）+ 18 鄉
- **fault lines**：
  - 鹿港歷史 vs 員林市現代（兩個彰化的不對稱）
  - 1723 設縣的「老彰化」vs 1812（Stage 0 預設）的時代誤差
  - 田尾公路花園 vs 荷蘭花卉的全球競爭
  - 客家飛地（田尾、永靖）vs 主流閩南
  - 縣治彰化市 vs 人口最多的員林市（行政 vs 經濟中心拉扯）
  - 「一府二鹿三艋舺」榮光 vs 鹿港今天的觀光化

### 想法跟感受

- **正面**：彰化人對「鹿港歷史 + 八卦山大佛 + 八堡圳」三大文化資產的在地驕傲 / 對 124 萬人全台第三大縣的人口自信 / 對員林雞腿便當 + 鹿港肉圓 + 田尾公路花園的在地飲食 + 產業
- **負面**：「鹿港 = 彰化」的全國框架疲乏 / 員林市區跟彰化市的縣治競爭尷尬 / 田尾公路花園被荷蘭搶風采 / 中部地震帶（雖然不是 921 震央但 921 也搖）
- **矛盾感**：「一府二鹿三艋舺」二鹿時代結束 200 年，鹿港還在生活 / 縣治彰化市 24 萬 vs 員林市 12 萬，兩個中心拉扯 / 1723 設縣是早於台南府的二級行政區之一，但全國認知是「彰化就是鹿港」

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 平埔族巴布薩族（西部平原）+ 洪雅族部分 千年居住
  - 17 世紀後漢人從鹿港 + 二林沿海開拓
  - 1709 八堡圳建立（康熙 48 年，二水起點）
  - 1717-1728 鹿港始見地圖 + 米倉建立
- **關鍵轉折**：
  - 1723 雍正元年彰化縣設立（從諸羅縣分出，含舊虎尾溪以北至大甲溪以南）
  - 1786 林爽文之役 + 鹿港龍山寺重建 + 「一府二鹿三艋舺」時代成形
  - 1788 鹿港天后宮建立（清乾隆 53 年）
  - 1812 設立淡水廳分支（非彰化縣設立年份，Stage 0 預設誤）
  - 1875 沈葆楨改革時期保留彰化縣（跟新竹縣 + 基隆同期）
  - 1895 日治：彰化廳 + 1920 改台中州彰化郡 + 員林郡 + 北斗郡 + 二林郡
  - 1922 彰化扇形車庫建成（蒸氣火車迴轉設備）
  - 1945 戰後：彰化縣設立（含原縣域）
  - 1961 八卦山大佛竣工（22m，6 層內部空間）
  - 1999 921 地震彰化縣受波及（雖非震央但全倒半倒不少）
  - 2000s 鹿港老街觀光化 + 員林市區擴張
- **當代意義**：彰化是台灣「中部走廊」+「二鹿榮光遺產」的典範縣。1723 設縣 + 鹿港「一府二鹿三艋舺」+ 八卦山大佛 1961 + 124 萬人全台第三大縣 — 四個故事撐起彰化的當代意義

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 彰化縣是台灣「最早設立的縣份之一」（1723 早於台南府縣大部分）
- 鹿港「一府二鹿三艋舺」是清領台灣最重要的港埠跟商業中心（18-19 世紀）
- 八堡圳 1709 是台灣最早的大型水利系統之一（早於八田與一 1930 嘉南大圳 221 年）
- 124 萬人是 22 縣市中人口第三大（次於新北 401 + 台中 285，比桃園 230 + 高雄 273 + 台南 186 都多）
- 對讀者：「為什麼一個 1723 設立 + 鹿港時代結束 200 年的縣，今天還是 124 萬人的人口大縣？」這個 frame 拆開後讓人看見「持續性」的意義

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：彰化平原（濁水溪 + 大肚溪沖積扇）是台灣第二大平原（次於嘉南平原約 4,550 km²，彰化平原約 1,200 km²）+ 八卦台地（八卦山西側）+ 大肚台地南端 + 西部沿海灘地 / 26 鄉鎮市分平原（彰化、員林、和美）vs 沿海（鹿港、福興、二林、芳苑）vs 台地（八卦山西側）
- **誰在這裡生活**：巴布薩族 + 洪雅族部分（已漢化）+ 1717 後漢人移民 + 客家飛地（田尾、永靖）+ 日治日本人 + 戰後外省（軍公教）+ 越南新住民
- **人和地有什麼共生**：濁水溪 → 西部米倉 / 沿海 → 王功蚵仔 + 鹿港漁業 / 八卦台地 → 八卦山大佛 + 賞鷹 / 田尾 → 公路花園（全台花卉中心）/ 員林 → 雞腿便當鐵道文化 + 員林楊桃 / 鹿港 → 龍山寺 + 天后宮 + 老街
