# Research Report: 新竹縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：客家擂茶／內灣老街（橫山鄉，老火車站、劉興欽漫畫博物館、野薑花肉粽）／北埔老街（金廣福公館、姜阿新洋樓）／湖口老街（三元宮、巴洛克紅磚街屋）／竹北高鐵站（縣治竹北市快速都市化）／新埔義民廟（客家信仰中心）／關西仙草／橫山尖石原民部落
- **世代差異**：
  - 戰前老世代知「竹東林業」（戰後阿里山、太平山、八仙山三大林場以外的北部林場）
  - 戰後 50-70s 世代知「煤礦業」（內灣、橫山、尖石煤礦支撐戰後經濟）
  - 80-90s 世代知「客家本色」（客家文化重點發展區政策受益地之一）+ 新竹科學園區擴張（從新竹市跨到寶山）
  - 千禧後世代知「竹北高鐵特區房價」（台積電員工宿舍 + 學區概念 + 客家年輕人買不起自己家鄉）
- **外人 vs 在地人**：外人覺得「客家庄體驗一下 + 看內灣老街」；在地人知「我們縣治竹北 8.8 萬人擠在 6 公里的科技園區外圍」「義民廟 235 年延續比台積電的 18 個月設計週期慢得多」「橫山尖石的原住民部落跟竹北高鐵特區是兩個時代」

### 多元面貌

- **主流敘事**：「客家庄縣」+「科技園區外圍」（兩個 frame 各自被簡化）vs 「沒有自己的城市感」（過度被新竹市/竹北市切割誤解）
- **支線敘事**：
  - 客家比例約 70%+（全台第二高，僅次苗栗），但縣內又跟賽夏族、泰雅族同住
  - 賽夏族（五峰鄉）vs 泰雅族（尖石鄉）兩大原住民族
  - 1718 王世傑率閩粵移民拓墾竹塹埔（含今新竹市 + 新竹縣）
  - 1812 設淡水廳 → 1875 改新竹縣（沈葆楨改革，跟基隆改名同期）
  - 1950 縣市分治：新竹市獨立省轄市，新竹縣治先在新竹市，1989 遷至竹北市
  - 1980 新竹科學園區成立 → 邊界跨新竹市 / 新竹縣 / 苗栗（不只在新竹市）
  - 新埔義民廟（清乾隆 53 年，1788）紀念林爽文之役客家義民 → 客家信仰中心
  - 13 鄉鎮市：竹北 / 竹東 / 新埔 / 關西 / 湖口 / 新豐 / 芎林 / 橫山 / 寶山 / 北埔 / 峨眉 / 五峰 / 尖石
- **fault lines**：
  - 客家庄 vs 科技園區（生活節奏 235 年慢延續 vs 18 個月設計週期）
  - 竹北市快速都市化 vs 其他鄉鎮（竹北房價漲翻 vs 山區鄉鎮人口外流）
  - 漢人客家 vs 原住民賽夏族 + 泰雅族（族群比例失衡）
  - 新竹縣 vs 新竹市 vs 苗栗北部（行政切割但生活圈一體）

### 想法跟感受

- **正面**：客家人對「義民信仰」「擂茶文化」有強烈認同 / 對北埔金廣福公館、湖口老街、內灣老街三個老街遺產的在地驕傲 / 客家委員會 2001 成立後文化政策受益最深的縣之一
- **負面**：「沒有自己的城市」感（竹北市的客家身分被快速都市化稀釋）/ 山區鄉鎮人口外流 / 房價漲翻讓客家年輕人買不起自己家鄉的家
- **矛盾感**：義民廟 235 年延續 vs 台積電 18 個月設計週期同時存在 / 縣治竹北 8.8 萬人擠在科學園區外圍 vs 五峰尖石山區人口外流

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 原住民道卡斯族（西部沿海平地）+ 賽夏族（五峰山區）+ 泰雅族（尖石山區）
  - 1718 王世傑率閩粵移民拓墾竹塹埔
  - 1788 新埔義民廟竣工紀念林爽文之役客家義勇
- **關鍵轉折**：
  - 1812 清領設淡水廳於竹塹（今新竹市區）
  - 1875 改新竹縣（沈葆楨改革，跟基隆改名同期）
  - 1895 日治：縣治竹塹，1920 廢廳置州郡屬新竹州
  - 1945 戰後：行政體制延續日治
  - 1950 縣市分治：新竹市獨立省轄市，新竹縣治留新竹市（暫）
  - 1980 新竹科學園區成立（跨新竹市/新竹縣/苗栗）
  - 1989 新竹縣治遷至竹北市
  - 2001 客家委員會成立 + 客家文化重點發展區政策起步
  - 2007 高鐵新竹站 + 竹北快速都市化
  - 2010s 台積電擴廠寶山 + 竹科員工湧入竹北
- **當代意義**：新竹縣是台灣「客家文化保存」+「科技產業擴張」+「原住民部落」三線同存的典範。竹北市的客家身分被快速都市化稀釋的張力是當代客家縣的縮影

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 新竹縣是台灣「客家庄圍繞科技園區」的物理見證（70%+ 客家比例 + 70 萬人 + 隔壁是台積電總部）
- 義民信仰是台灣客家社群最具識別度的宗教文化（新埔義民廟 1788 起 + 中元義民節）
- 內灣老街、北埔老街、湖口老街三個保存最完整的客家老街
- 賽夏族（五峰）跟泰雅族（尖石）的山區部落跟竹北的高樓並存，地理切割但文化交織
- 對讀者：「為什麼一個客家比例 70%+ 的縣，會被縣治竹北的科技化代表？」這個 frame 拆開後讓人重新看見客家如何在 18 個月設計週期中保持 235 年傳統

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：頭前溪沖積平原（西部沿海）+ 雪山山脈西側丘陵（五峰、尖石山區）/ 海岸線：新豐、新埔、湖口 / 內陸：芎林、橫山、寶山、北埔、峨眉 / 山地：五峰、尖石
- **誰在這裡生活**：道卡斯族（沿海平地，已漢化）/ 賽夏族（五峰鄉）/ 泰雅族（尖石鄉）/ 18 世紀廣東客家移民 / 戰後外省移民（軍公教 + 竹科員工）/ 2000s 後台積電擴廠湧入的技術新移民
- **人和地有什麼共生**：丘陵 → 茶葉（東方美人 + 椪風茶 + 北埔擂茶）/ 山林 → 林業（戰前竹東林場）+ 油桐花 / 客家庄 → 義民廟 + 紅龜粄 + 客家小炒 / 山區部落 → 賽夏族矮靈祭（跨苗栗共辦）+ 泰雅族傳統 / 平地 → 科技園區擴張

### 切入點清單（候選 hooks）

1. **「義民廟 235 年 vs 台積電 18 個月」**（1788 新埔義民廟竣工 → 客家信仰核心 → 跨越清領、日治、戰後、民主化、科技化五個時代 vs 竹北高鐵特區 8.8 萬人擠在台積電寶山廠的 18 個月設計週期外圍）
2. **「客家庄圍繞科學園區」的不對稱**（70%+ 客家比例縣 + 縣治竹北是科技人口湧入 + 五峰尖石原住民鄉外流 - 三個世界同一個縣）
3. **內灣 + 北埔 + 湖口三個客家老街**（三個都有完整保存：內灣鐵道 + 北埔金廣福 + 湖口巴洛克紅磚 → 客家老街生態系不是單點）
4. **賽夏族矮靈祭 + 泰雅族尖石部落**（五峰鄉跟苗栗南庄共辦 + 尖石泰雅族部落在科技園區隔壁）
5. **1989 縣治從新竹市遷至竹北市**（行政切割 1950 + 縣治遷移 1989 → 兩個時代的「新竹縣意識」斷代）
6. **金廣福公館 1835 + 北埔老街**（姜秀鑾 + 周邦正合股「金廣福」開墾大隘 → 北埔成形 → 戰後北埔擂茶觀光化）
7. **王世傑 1718 拓墾竹塹埔**（最早的閩粵漢人拓墾紀錄 → 比新竹建城 1718 更早 → 新竹縣的歷史起點）

### 預期核心矛盾候選（待 Stage 1.4 收斂為 ≤ 30 字）

- A：義民廟 235 年慢延續 vs 台積電 18 個月設計週期同住
- B：客家比例 70%+ 的縣，縣治竹北的客家身分被科技化稀釋
- C：科技園區員工買的房子，讓客家年輕人買不起自己家鄉
- D：竹北 8.8 萬擠科技園區外圍，五峰尖石山區人口外流 — 一個縣兩個方向
