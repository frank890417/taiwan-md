# Research Report: 花蓮縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：太魯閣國家公園／立霧溪／清水斷崖／七星潭海岸／0403 地震（2024/04/03 規模 7.2，台北 101 都搖）／慈濟靜思精舍（1966 證嚴法師）／蘇花公路 9 彎 18 拐／蘇花改 2020 通車／玉里布農族／豐田移民村／鳳林客家／太魯閣族（2004 正名）／撒奇萊雅族（2007 正名）／加禮宛事件 1878
- **世代差異**：
  - 戰前老世代知「奇萊」（古名 + 1875 沈葆楨開山撫番 + 加禮宛事件 1878 後設立宜蘭花蓮通道）
  - 戰後 50-70s 世代知「北迴鐵路 1980 通車」（基隆 → 花蓮 18 小時變 5 小時）+ 大理石產業
  - 80-90s 世代知「太魯閣國家公園 1986 設立」+「玉里地震」（1951 規模 7.3 + 1986 規模 6.8）
  - 千禧後世代知「0403 地震 2024」+ 太魯閣坍方 + 觀光業重挫
- **外人 vs 在地人**：外人覺得「太魯閣 + 七星潭 + 慈濟」就走人；在地人知「我們 132 公里長的縣山海距離最近處只有 5 公里」「0403 地震一秒鐘把太魯閣崩了，但我們 50 年前就在學跟地震一起住」「玉里的布農族是 1937 從南投搬下來的」「撒奇萊雅族 2007 才正名，之前 129 年被當阿美族」

### 多元面貌

- **主流敘事**：「東部後花園」+「太魯閣觀光天堂」（過度浪漫化）vs 「地震災區」（過度悲觀）
- **支線敘事**：
  - **5 個原住民族同住**：阿美族（南勢阿美 + 海岸阿美，最大族群）+ 太魯閣族（2004 從泰雅族分出，秀林、萬榮、卓溪部分）+ 撒奇萊雅族（2007 正名，加禮宛事件後潛藏成阿美族 129 年）+ 賽德克族（部分）+ 布農族（玉里 1937 從南投遷來）
  - **0403 地震 2024/04/03 07:58**：規模 7.2 / 震央壽豐鄉 / 深 19.7km / 秀林鄉和平里震度 6 強 / 太魯閣國家公園估復原 7 年 / 觀光客 21 萬（2024）vs 數百萬正常
  - **1878 加禮宛事件**：噶瑪蘭族 + 撒奇萊雅族聯合抗清 → 戰敗後撒奇萊雅族潛藏成阿美族 129 年 → 2007 正名
  - **1937 玉里布農族遷徙**：從南投山區（卡社、丹大、巒、郡、卓社）大舉遷往玉里跟卓溪
  - **慈濟靜思精舍 1966 證嚴法師創立**：花蓮新城鄉 + 1991 慈濟醫院 + 至今最大 NGO + 國際慈善
  - **北迴鐵路 1980 通車**：基隆 → 花蓮從船運/客運 18 小時變鐵路 5 小時
  - **太魯閣國家公園 1986 設立**：日治時期的「次高太魯閣國立公園」之延續
  - **蘇花改 2020 全段通車**：取代蘇花公路 9 彎 18 拐 + 蘇花安計畫
  - **大理石產業**：花蓮縣礦業重鎮 + 戰後輸出顛峰
  - **豐田移民村**：1913 日治時期日本人移民村（吉野村、豐田村、林田村）+ 戰後本省人接收
  - **鳳林客家**：花東縱谷北段的客家庄
- **fault lines**：
  - 5 族原住民跟漢人移民的歷史張力（加禮宛事件 1878 / 太魯閣事件 1914 / 布農族遷徙 1937）
  - 太魯閣族（2004） + 撒奇萊雅族（2007）正名運動的當代意義
  - 0403 地震後觀光重挫 vs 在地原住民部落復原
  - 慈濟（國際宗教 NGO） vs 花蓮原住民傳統 vs 觀光商業
  - 蘇花公路（傳統）vs 蘇花改（2020）vs 蘇花安（計畫中）的交通演進

### 想法跟感受

- **正面**：花蓮人對「山海之間」「太魯閣是我們的」有強烈認同 / 對 5 族同住 + 慈濟 + 鳳林客家的多元自信 / 對 0403 地震後韌性的肯定
- **負面**：「地震災區」「觀光重挫」的失落 / 撒奇萊雅族被當阿美族 129 年的歷史傷口 / 加禮宛事件 + 太魯閣事件的殖民暴力記憶
- **矛盾感**：山海距離最近處只有 5 公里 vs 132 公里長的縣 / 0403 地震一秒崩太魯閣 vs 50 年前就在學跟地震住 / 慈濟靜思精舍背景是花蓮 1960s 連醫院都沒有

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 阿美族（南勢、海岸） + 太魯閣族 + 撒奇萊雅族 + 賽德克族 + 布農族（後遷入）千年居住
  - 17 世紀後西班牙、荷蘭短暫接觸（聖薩爾瓦多城 + 北部荷蘭時期）
  - 1875 沈葆楨開山撫番 → 蘇花古道
- **關鍵轉折**：
  - 1878 加禮宛事件：噶瑪蘭族（從宜蘭遷來）+ 撒奇萊雅族聯合抗清 → 戰敗 + 撒奇萊雅族 129 年潛藏
  - 1895 日治：花蓮港廳 + 日本移民村政策 + 1913 吉野村、豐田村、林田村
  - 1914 太魯閣事件：日軍討伐太魯閣族（佐久間左馬太總督主導）
  - 1937 布農族大舉從南投遷往玉里、卓溪（皇民化日治政策）
  - 1945 戰後：花蓮縣設立 + 行政體制延續日治
  - 1951 玉里地震規模 7.3（縱谷大地震）
  - 1966 證嚴法師創立慈濟靜思精舍（花蓮新城）
  - 1980 北迴鐵路通車（基隆 → 花蓮 5 小時）
  - 1986 太魯閣國家公園設立
  - 1991 慈濟醫院開業
  - 2004 太魯閣族正名（從泰雅族分出）
  - 2007 撒奇萊雅族正名（從阿美族分出，129 年後恢復獨立族別）
  - 2020 蘇花改全段通車
  - 2024/04/03 0403 地震規模 7.2 → 太魯閣國家公園估復原 7 年
- **當代意義**：花蓮縣是台灣「山海之間最薄的縣」+「5 族原住民同住」+「正名運動最深」的典範。0403 地震是 1999 921 之後台灣最大內陸地震，太魯閣國家公園可能 7 年才完整復原

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 花蓮縣是台灣「山海之間最薄的縣」（132 公里長 + 中央山脈到太平洋最近處只有 5 公里）
- 5 族原住民同住 + 太魯閣族 + 撒奇萊雅族正名運動最深（2004 + 2007）
- 加禮宛事件 1878（撒奇萊雅族 129 年潛藏歷史傷口）+ 太魯閣事件 1914（日軍討伐）的殖民暴力記憶
- 慈濟靜思精舍 1966 在這裡誕生，背景是花蓮 1960s 連完整醫院都沒有
- 0403 地震 2024 是 1999 921 之後台灣最大內陸地震 + 太魯閣國家公園估復原 7 年
- 對讀者：「為什麼台灣最美的地景在最常被地震的縣？」這個 frame 拆開後讓人重新看見「美 vs 災」的物理共生

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：菲律賓海板塊撞歐亞板塊形成中央山脈 + 海岸山脈 + 花東縱谷 / 132 公里長南北延伸 / 山海距離最近處 5 公里（清水斷崖）/ 立霧溪 + 木瓜溪 + 秀姑巒溪 + 卑南溪沖積平原 / 地震帶活躍（菲律賓海板塊隱沒）
- **誰在這裡生活**：阿美族（南勢、海岸，最大族群）+ 太魯閣族（秀林、萬榮、卓溪部分）+ 撒奇萊雅族（新城、瑞穗一帶）+ 賽德克族（部分萬榮）+ 布農族（玉里、卓溪，1937 後遷來）+ 平埔族噶瑪蘭族（1878 加禮宛事件後遷來部分留下）+ 漢人客家（鳳林）+ 漢人閩南 + 日治日本移民後代少數
- **人和地有什麼共生**：太平洋 → 七星潭海岸 + 海岸阿美族傳統 / 中央山脈 → 太魯閣立霧溪 + 太魯閣族 / 縱谷 → 玉里米 + 富里米 + 布農族 + 客家鳳林 / 海岸山脈 → 撒奇萊雅族 + 阿美族祭典 / 地震 → 大理石產業（地質景觀）+ 0403 重挫

### 切入點清單（候選 hooks）

1. **0403 地震 2024 那一秒崩了太魯閣**（07:58 規模 7.2 + 太魯閣國家公園估復原 7 年 + 觀光客 21 萬 vs 數百萬，但花蓮人 50 年前就在學跟地震住，1951 玉里地震 7.3 + 1986 規模 6.8）

3. **太魯閣族 2004 從泰雅族分出**（百年被歸類為泰雅族 → 2004 獨立成族 + 太魯閣的命名爭議）
