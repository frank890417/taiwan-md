# Research Report: 金門縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：1949/10/25 古寧頭戰役（56 小時擊敗共軍登陸）／1958/08/23 八二三砲戰（44 天 47 萬發砲彈）／「單打雙不打」騷擾砲擊（1958-1979 直到中美建交）／金門高粱酒（1952 金門酒廠成立 / 胡璉收編「九龍江酒廠」）／翟山坑道／太武山「毋忘在莒」／莒光樓／風獅爺／戰備坑道網絡／模範街（清末民初洋樓）／聚落古厝
- **世代差異**：
  - 戰前老世代知「明清海防 + 鄭成功反清復明」（1646 鄭成功在金門廈門起兵）
  - 戰後 50-90s 世代知「戰地」（1956-1992 戰地政務 36 年 + 古寧頭 + 八二三 + 單打雙不打）
  - 90s-2000s 世代知「兩岸對峙的金門」+「金門高粱酒」（國軍 5 萬 vs 居民 14 萬比例）
  - 千禧後世代知「2001 小三通開放」+「戰地觀光化」+「金門大學 1997」+ 廈門對望（最近 1.8km，比距台北 358km 近）
- **外人 vs 在地人**：外人覺得「金門看坑道 + 喝高粱 + 看風獅爺」就走人；在地人知「我們離廈門最近處只有 1.8km 但是台灣的縣」「八二三砲戰 44 天打了 47 萬發砲彈 + 單打雙不打又 21 年」「金門 2001 小三通開放後跟廈門經濟連結比跟台灣本島深」「我們是福建省金門縣，跟連江縣（馬祖）同樣戰後分治的縣」

### 多元面貌

- **主流敘事**：「戰地觀光的金門」（過度浪漫化）vs 「冷戰前線縣」（過度悲觀）
- **支線敘事**：
  - **1949/10/25-27 古寧頭戰役**：共軍三個團 9,000+ 人登陸古寧頭 + 國軍 56 小時激戰 + 殲俘 15,000+ → 決定金門 75 年命運的 56 小時
  - **1958/08/23 八二三砲戰**：共軍 47 萬發砲彈 + 44 天激戰 + 國軍 17 萬人在金門 + 後續「單打雙不打」騷擾砲擊到 1979/01/01 中美建交為止（21 年）
  - **1952 金門酒廠**：防衛司令官胡璉收編民間「九龍江酒廠」+ 葉華成釀酒師 + 金門高粱酒全台知名
  - **1956-1992 戰地政務 36 年**：跟連江縣（馬祖）同步戰地政務 + 軍法審判 + 出入境管制 + 國軍 vs 居民比例 5 萬 vs 14 萬（粗估）
  - **2001/01/01 小三通開放**：金門 - 廈門 + 馬祖 - 福州 直航 + 跟廈門經濟連結加深
  - **6 鄉鎮**：金城鎮（縣治）+ 金湖鎮 + 金沙鎮 + 金寧鄉 + 烈嶼鄉（小金門）+ 烏坵鄉（屬金門縣但跟連江縣鄰近，特殊行政結構）
  - **福建省金門縣** vs 福建省連江縣（馬祖）：兩個外島縣同屬中華民國福建省（非台灣省）
  - **戰地遺址觀光化**：翟山坑道 + 太武山「毋忘在莒」蔣公題字 + 莒光樓 + 古寧頭戰史館 + 八二三戰史館
  - **金門大學 1997 成立**：戰地轉教育 + 跟在地居民的關係
- **fault lines**：
  - 古寧頭 + 八二三 + 單打雙不打的軍管 36 年歷史 vs 今天觀光化
  - 距廈門 1.8km vs 距台北 358km 的物理矛盾
  - 2001 小三通後金門 vs 廈門經濟連結 vs 跟台灣本島
  - 福建省金門縣 vs 福建省連江縣（馬祖）的兩個外島縣 vs 台灣本島
  - 戰地遺產（坑道 + 砲戰）vs 在地民俗（風獅爺 + 古厝聚落）
  - 金門大學 1997 + 金門高粱（戰地經濟）vs 在地年輕人外流

### 想法跟感受

- **正面**：金門人對「最前線」「冷戰物理見證」有強烈自尊（古寧頭 + 八二三決定了兩岸分治 75 年）/ 對金門高粱 + 模範街 + 風獅爺的在地驕傲
- **負面**：「戰地」標籤的疲乏 / 距廈門 1.8km vs 距台北 358km 的歸屬感矛盾 / 軍管 36 年的歷史傷口（跟馬祖同樣晚解嚴 4 年）/ 2001 小三通後跟廈門經濟連結 vs 台灣意識的拉扯
- **矛盾感**：離廈門最近的縣是台灣的縣 / 戰地觀光把砲戰歷史包裝成 IG 打卡 / 1949 古寧頭那 56 小時決定了金門 75 年命運 / 福建省金門縣是台灣的縣

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 唐朝牧馬地 + 五代後周顯德 6 年（959）設立晉江縣
  - 明初海防要塞 + 1387 周德興建金門所城（明洪武 20 年）
  - 1646 鄭成功在金門廈門起兵反清
- **關鍵轉折**：
  - 1949/05 國共內戰結束前夕，國軍從大陸撤退至金門
  - 1949/10/25-27 古寧頭戰役（共軍登陸 + 國軍 56 小時擊敗）
  - 1952 金門酒廠成立（胡璉收編九龍江酒廠 + 葉華成）
  - 1956 戰地政務開始（跟馬祖同步）
  - 1958/08/23-10/05 八二三砲戰（44 天 47 萬發 + 後續「單打雙不打」到 1979）
  - 1979/01/01 中美建交 + 砲擊完全停止
  - 1991/05/01 動員戡亂時期臨時條款廢止 → 1992/11/07 戰地政務廢止（跟馬祖同步）
  - 1992-1997 民選縣長 + 金門大學 1997 成立
  - 2001/01/01 小三通開放（金門-廈門 + 馬祖-福州）
  - 2010s-2020s 戰地觀光化 + 跟廈門經濟連結
- **當代意義**：金門是台灣「冷戰前線變觀光景點 + 兩岸分治物理見證」的典範。古寧頭 + 八二三的 56 小時 + 44 天決定了兩岸分治 75 年，但今天小三通 24 年後金門跟廈門的經濟連結比跟台北深

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 金門是台灣「主權邊界」的物理見證（距廈門 1.8km vs 距台北 358km / 福建省金門縣）
- 1949 古寧頭戰役決定了兩岸分治 75 年（如果共軍成功登陸金門，後續歷史完全不同）
- 1958 八二三砲戰是冷戰熱戰的最具體案例（44 天 47 萬發砲彈 + 21 年單打雙不打）
- 2001 小三通是兩岸開放最早的試點（金門 - 廈門直航 + 馬祖 - 福州直航）
- 對讀者：「為什麼一個離廈門 1.8km 的縣，是台灣的縣？」這個 frame 拆開後讓人看見冷戰、戒嚴、兩岸分治的物理現實 — 跟馬祖（連江縣）形成兩個外島縣的對話

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：金門島（本島）+ 烈嶼（小金門）+ 大膽 + 二膽 + 烏坵（外島中的外島）+ 多個衛星小島 / 花崗岩地質（跟馬祖同源閩東花崗岩）/ 海岸線複雜 + 軍事工事密集 / 距大陸最近處 1.8km（廈門角嶼）+ 距台北 358km
- **誰在這裡生活**：閩南漢人移民（明朝起，跟廈門同源）+ 1949 後國軍 + 軍管時期 5 萬國軍 + 解嚴後年輕人外流 + 2001 後跟廈門經濟連結的兩岸跨境工作者
- **人和地有什麼共生**：花崗岩 → 古厝聚落（模範街 + 山后聚落）+ 風獅爺民俗 / 戰略位置 → 古寧頭 + 八二三戰役 + 戰備坑道 + 太武山軍事工事 / 海洋 → 高粱酒（戰後 1952 起）+ 鱟魚生態 + 跟廈門海域共享 / 戰地觀光 → 翟山坑道 + 莒光樓 + 古寧頭戰史館
