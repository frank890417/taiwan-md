# Taiwan.md の起源物語

> **30 秒の概要：** Taiwan.md は、2026 年 3 月に台湾のニューメディア・アーティスト **呉哲宇（Che-Yu Wu）** が立ち上げたオープンソースの台湾知識データベース。歴史・文化・食・テクノロジー・自然など 12 の主題領域を、中英二言語、CC BY-SA 4.0 ライセンスで完全にオープンソース化。立ち上げの動機は、2024 年ヴェネツィア・ビエンナーレで一人のイタリア人キュレーターが投げかけた一言の問いだった：「Where can I learn about Taiwan? Like, really learn?」このウェブサイトは、その問いから二年後の答えである。
>

## ビエンナーレでのあの問い

2024 年春。第 60 回ヴェネツィア・ビエンナーレ、Personal Structures 部門。

台湾のニューメディア・アーティスト呉哲宇は、自身の作品《ソウルフィッシュ（靈魂魚）》で参加していた。オープニング・レセプションで、一人のイタリア人キュレーターが歩み寄ってきて、こう尋ねた：

> **「Where can I learn about Taiwan? Like, really learn?」**

彼はその場で答えを持たなかった。

もちろん、台湾に関する資料がないわけではない。Wikipedia のエントリー、ロンリープラネットのガイド、学術論文――どれもある。しかし本気でこの島を「学びたい」外国人にとって、台湾を主体にした、温度があり、深く、継続的に更新される入口は存在しなかった。

さらに核心的な問題がある――**AI がインターネットから「台湾とは何か」を学ぶとき、誰の台湾を学んでいるのか？**

台湾に関する高品質な英語コンテンツが主に非台湾視点から生産されているなら、AI が学ぶ「台湾」は、ここに住む人々が認識する台湾ではない。

これはアイデンティティ政治の問題ではない。**知識主権の問題**である。

あの問いは、彼の中に二年間埋められていた。

## なぜ Markdown なのか

2026 年 3 月、彼は Taiwan.md を立ち上げた。

名前そのものが答えである：最もシンプルで、最もオープンで、AI の学習モデルに最も読み取られやすいフォーマットで、台湾を書く。

- **.md** は Markdown の拡張子――プレーンテキスト、どの AI、クローラー、エディターでも直接読める
- **CC BY-SA 4.0 ライセンス**、完全にオープンソース、複製・翻訳・再利用可能
- **GitHub での協働**、誰でも PR を提出して補完・修正・拡張できる
- **中英二言語優先**（後に日本語・韓国語・フランス語にも拡張）

これは観光サイトでも百科事典でもない。**台湾のデジタル人類学博物館**である。

## これは一人のプロジェクトではない

立ち上げ以来、Taiwan.md は 900 以上の GitHub star、100 以上の国からの訪問者、数十人の貢献者による PR を受けている。臺史博物館長**張隆志**が公に支持し、**オードリー・タン**が国際的なネットワークに引き合わせた。Portaly スポンサー・プラットフォームは立ち上げ第一月目に開通した。

各記事の contributors 欄には共同執筆者の名前が記載されている。この「博物館」の真の姿は、**一人の作品集ではなく、「台湾を一度ちゃんと語りたい」と願う人々が集まり共同で書き続けている、生きて呼吸するオープンソースの生態系**である。
